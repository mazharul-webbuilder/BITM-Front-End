There is 5 positioning system in CSS

  1. Static >>This is by default in every html tag in a web page.
  2. Relative >>By relative we can change a element anywhere in a webpage with 4 css property.
    1. top
    2. bottom
    3. left
    4. right
    Note: Relative change his positin with his responding area.
  3. Absolute >>Absolute change his positin with his parent responding area
  4. Fixed >>Fixed Never Changed his position.
  5. Sticky >>This is most important and loveable position, check the example below.
